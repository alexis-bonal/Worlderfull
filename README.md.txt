                                Description
Dans le monde de Worlderfull, les chammés s’amusent à régulièrement voler les reliques du savoir. 
Afin de les récupérer, le joueur devra parcourir les différentes régions et relever tous les défis.
Le mont Maths est réputé pour être difficilement accessible, là haut, il faut bien calculer son itinéraire.
Le bois des Mots Perdus est habité par la tribu des Mokachés avec qui il faudra savoir parler pour ne pas se perdre.
On raconte que dans la vallée Historique les souvenirs du monde passé ressurgissent.
Au port Anglo, des navires de tout Worlderfull échangent leur marchandises dans la langue anglaise. 
C’est dans ces lieux que se trouvent les reliques du savoir.

Les jeux :
 - Français :
    - intrus
    - orthographe des couleurs
    - synonyme

 - Mathématique
    - calcul mental
    - égalités de calcul
    - séparer dizaines et unités
    - Trie de nombres (croissant décroissant pair impair)     
    
 - Histoire-Géographie
    - Grandes villes Françaises
    - Personnages célèbres
    - Les Fleuves
    - Les Régions
    - Phrise chrono

 - Anglais
    - Traduction de mots
    - Donner les saisons
    - Donner les jours
    - Donner les mois
    - Les couleurs

Comment jouer au jeux:
1- Situer vous dans le repertoire Worlderfull
2- Compiler le jeux en executant la commande ./compile.sh,un message de reussite de compilation ou d'echec s'affichera.
3- Lancer le jeux avec la commande ./run.sh(Possibiliter de lancement+ execution du jeux avec la commande ./compile_run.sh
4- Lancement du jeux avec un deffilement Ascii art affichant le nom du jeux
5- Arriver dans le gestionaire de sauvgarde ou vous saisirer votre pseudo pour le jeux.(Si une parti a deja ete cree on peux la recuperer)
6- Apres creation de la partie s'affiche l'introduction a l'histoire du jeux.
7- Arriver dans le monde de Worlderfull avec les differentes regions et la possibilité d'acceder au parametre et a la sorti du jeux.
8- Pour ce deplacer il suffit de taper la lettre indique.
9- Apres choix de la regions arrive dans le menus de la region avec le meme principe de deplacement.
10- Choix de l'epreuve a realiser.
11- Realiser l'epreuve choisie en suivant la consigne donnée
12- Si reussite ou echec de l'epreuve un message s'affichera.
13- Une fois l'epreuve fini et validé elle s'eclaire de couleure jaune ce qui permet de suivre la progression du joueur dans le jeux.
14- Apres la reussite de toute les epreuves de la region celle si s'eclairera en jaune.
15- Apres la reussite de toute les epreuve de toute les régions, en quittant le jeux le joueur vera afficher sur son ecrant une lettre du Roi de Worderfull le grand Roi Rathur.
16- En quittant le jeux un message de remerciment s'affiche.
